# serious-http-library
For serious programmers.

# About

## v1

Serialization and deserialization of HTTP/1.1 accoring to [RFC9112](https://www.rfc-editor.org/rfc/rfc9112.html).

## v2

Serialization, deserialization and session of HTTP/2 accoring to [RFC9113](https://www.rfc-editor.org/rfc/rfc9113.html).

## URI

Serialization and deserialization of URI according to RFC3986.
